# セキュリティ

## サポート対象のバージョン

学習・参考用のサンプルコードのため、特定バージョンの長期サポートは行っていません。可能な限り依存パッケージを更新してください。

## 脆弱性の報告

シークレットや認証情報は **Issue や Pull Request に貼らないでください。**

脆弱性を見つけた場合は、リポジトリの **GitHub Security Advisories** から報告するか、メンテナが案内する連絡手段のみを利用してください。

## 公開前のチェックリスト（参考）

- `.env`・`aws-exports.js`（実値）がリポジトリに含まれていないこと
- Google Maps・LINE・AWS などの **キー・トークンを画面表示・ログ出力していないこと**
- 本番で使うシークレットは **環境変数やシークレット管理サービス** から注入すること
